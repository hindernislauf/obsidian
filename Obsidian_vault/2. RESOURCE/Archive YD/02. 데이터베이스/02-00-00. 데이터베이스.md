# DB == Database
- 데이터를 통합하여 관리하는 데이터의 집합체.
- 여러 사람이나 프로그램이 데이터를 쉽게 공유하기 위해 체계적으로 관리되는 데이터의 집합!


# DBMS == Database Management System
- 데이터베이스를 관리하는 미들웨어 시스템을 DBMS라 칭함.
- 크게 RDBMS(관계형: SQL DB)와 나머지(NO SQL, 검색 엔진 등)으로 나뉨
